About the Project

KRONOS is an Arduino-based project that is powered by the UNO R3 module, it is a mechanical clock with a digital typeface/layout.


  • Majority of the physical components of the design uses acrylic.

  • It follows the 24-hour format to tell the time.

  • Has an RTC clock module to keep track of time
